# KD Tech Test - Logic Assessment

## Syarat

Pastikan sudah install **Node.js** dan **npm**.

## Cara Instalasi

Sebelum menjalankan unit test, harus menginstal seluruh *dependencies* yang diperlukan:

```bash
npm install
```

## Cara Menjalankan Unit Test

Setelah proses instalasi selesai, jalankan unit testing dengan command:

```bash
npm test
```
